# Favicon Setup

This folder contains all favicon and app icon files for GrooveSheet.

## Files Generated

### Main Favicon
- **favicon.ico** - Multi-resolution ICO file (16x16, 32x32, 48x48) for desktop browsers

### PNG Icons (icons/ folder)
- **favicon-16x16.png** - Small browser tab icon
- **favicon-32x32.png** - Standard browser tab icon
- **favicon-48x48.png** - Large browser tab icon
- **apple-touch-icon.png** (180x180) - iOS home screen icon
- **android-chrome-192x192.png** - Android home screen icon (low-res)
- **android-chrome-512x512.png** - Android home screen icon (high-res)

### Manifest
- **site.webmanifest** - PWA manifest file referencing the Android icons
